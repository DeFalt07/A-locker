To covert this python file into a executable file you need to install a file from command prompt to do that follow these steps 
open cmd - type
pip install pyinstaller

after installing shift+right click on the 'Helper-Timelapse' window run with power shell 
then write these commands 
pyinstall -w -F --onefile --icon=icon.ico timelapse_assist.py

Then wait for few seconds and wallah!! your done..
